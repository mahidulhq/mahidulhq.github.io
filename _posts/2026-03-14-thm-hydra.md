**Flag 01:** For brute-force web password we must know which type of request it is making. GET or POST methods are commonly used and then by follwing we use the commands 
`sudo hydra <username> <wordlist> MACHINE_IP http-post-form "<path>:<login_credentials>:<invalid_response>"`
